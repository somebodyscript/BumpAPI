BumpAPI Documentation

Introduction

BumpAPI is a powerful Python library designed to interact with external APIs of websites. It facilitates easy extraction of data, navigation, and interaction with web elements. This library streamlines the process of web scraping, enabling users to retrieve and manipulate specific content from websites effortlessly.

Usage

To start using the BumpAPI library, follow these steps:

  1. Ensure you have Python installed on your system. You can download it from here.
  2. Install BumpAPI using pip by running the following command in your terminal or command prompt:
    pip install bumpAPI
  3. Once installed, import the library in your Python code:
    from bumpAPI import BumpAPI
  4. Create an instance of the library by initializing bump = BumpAPI()
  5. Set the target URL using the bump.target() method to specify the website you want to interact with.
  6. Execute the target with the bump.executor() method to fetch the webpage content.
  7. Start using the various methods available in the BumpAPI library through the bump object to interact with the webpage, extract data, and perform actions.

Refer to the library documentation for detailed information on available methods and their usage.

Get started

To begin using the BumpAPI library, follow these basic commands:

  1. Set Target URL: Use bump.target('URL') to set the target website to interact with.
  2. Execute Target: Run bump.executor() to fetch the webpage content.
  3. View Page Text: Retrieve the entire page text using bump.text().
  4. Find Text: Search for a specific text using bump.find('text_to_find').
  5. Cut Text: Extract a portion of text between two keywords using bump.cut('start_keyword', 'end_keyword').
  6. Click Element: Simulate a click action on an element using bump.click('element_id').
  7. Send Text: Input text into the webpage using bump.send('your_text').

For example:

bump.target('https://example.com')
bump.executor()
bump.text()
bump.find('search_text')
bump.cut('start_keyword', 'end_keyword')
bump.click('element_id')
bump.send('your_text')

Refer to the library documentation for advanced commands and usage instructions.

Library Commands

bump.target('URL')

Description: Sets the target URL for interacting with the website.

Usage:

bump.target('https://example.com')

bump.executor()

Description: Fetches the webpage content from the specified URL.

Usage:

bump.executor()

bump.text()

Description: Retrieves and displays the entire text content of the webpage.

Usage:

bump.text()

bump.find('search_text')

Description: Searches for the specified text within the webpage.

Usage:

bump.find('example text')

bump.cut('start_keyword', 'end_keyword')

Description: Extracts text between two specified keywords.

Usage:

bump.cut('start', 'end')

bump.click('element_id')

Description: Simulates a click action on a specific HTML element by its ID.

Usage:

bump.click('submit_button')

bump.send('your_text')

Description: Inputs text into the webpage, simulating user input.

Usage:

bump.send('Your message here')

bump.wait(seconds)

Description: Pauses execution for the specified number of seconds.

Usage:

bump.wait(5)

bump.refresh('type')

Description: Refreshes the webpage content. 'type' specifies a full refresh.

Usage:

bump.refresh('type')

bump.links()

Description: Retrieves and displays all the hyperlinks on the webpage.

Usage:

bump.links()

bump.images(download=True)

Description: Retrieves all image URLs on the webpage. 'download=True' downloads the images.

Usage:

bump.images(download=True)

bump.summary()

Description: Displays a summarized version of the page content.

Usage:

bump.summary()

bump.stats()

Description: Provides statistics of HTML tag occurrences on the webpage.

Usage:

bump.stats()

bump.headers()

Description: Displays the headers of the target URL.

Usage:

bump.headers()

bump.compare('target_url')

Description: Compares the content of two URLs.

Usage:

bump.compare('https://example.com')

bump.extract('title')

Description: Extracts HTML elements based on their title.

Usage:

bump.extract('Page Title')
